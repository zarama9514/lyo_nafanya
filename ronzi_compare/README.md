# Сравнение модели с экспериментом Ronzi et al. (2003), Factor VIII/IX

Здесь наша модель первичной сушки (`../lyo_model.py`) сопоставляется с
**оптимизированным циклом из статьи Ronzi et al., *Optimisation of a freeze-drying
process of high purity Factor VIII and Factor IX concentrates*, Chem. Eng. Process.
42 (2003) 751–757** (их Fig. 5 и Табл. 4).

Цель — проверить, воспроизводит ли модель реальный цикл по **времени**, **форме**
кривой температуры продукта и **отсутствию коллапса**, и понять, откуда берутся
расхождения.

Запуск:
```bash
uv run python compare_ronzi.py     # -> ronzi_vs_model.png
```

---

## 1. Что воспроизводим (условия Ronzi, Табл. 4)

| Стадия | Плита | Давление | Время |
|---|---|---|---|
| Заморозка | −70 °C (set), −3 °C/мин | атм. | ~1 ч |
| Первичная сушка | −30 → −15 (4 ч), затем → −5 (4 ч) | ~20 Па (0.15 Torr) | ~9 ч |
| Вторичная сушка | −5 → +25 (10 ч) | ~16 Па | ~10 ч |

Дополнительно: Tim(FVIII) = −9 °C; наблюдения Ronzi — **коллапс при продукте −20 °C**,
целостность при −35 °C. Флакон 1 мл, налив 1 см → Ap ≈ 1 см².

---

## 2. Что реально считает модель, а что дорисовано (ВАЖНО, честно)

Наша модель симулирует **только первичную сушку** (сублимацию льда). Поэтому:

| Часть графика (панель A) | Источник | Расчёт модели? |
|---|---|---|
| Заморозка (провал переохлаждения, отскок) | кинематическая схема (кусочные функции) | ❌ нет |
| **Первичная сушка** (синяя кривая в зелёной зоне) | `primary_drying()` | ✅ **да** |
| Скачок продукта к плите в конце | экспонента (модель обрывается на конце сушки) | ❌ нет |
| Вторичная сушка | кинематическая схема | ❌ нет |
| Линия плиты | заданный **вход** (рампа по Табл. 4) | вход, не выход |

**Панель B** — целиком расчёт `primary_drying()` (обе кривые продукта); плита —
заданный вход; маркеры Ronzi/Tim/порога — референсы.

Иными словами: физикой посчитана **только первичная сушка**; заморозка и вторичная
сушка на панели A пририсованы для узнаваемости формы Fig. 5.

## 3. Формулы и дополнения, использованные здесь

Базовая физика — из корневого `../lyo_model.py` (см. корневой `../README.md`,
§3–4): подвижный фронт (Стефан), связанная тепло-массопередача, кнудсеновское
сопротивление, критерий коллапса. Здесь добавлено/задано **специфично под Ronzi**:

**(а) Рампа температуры полки** (вместо постоянной) — `Ts_primary(t)`:
```
Ts(t) = -30 + (15/4)·t            при t < 4 ч
        -15 + (10/4)·(t-4)        при 4 ≤ t < 8 ч
        -5                        при t ≥ 8 ч
```

**(б) Сопротивление сухого слоя — кнудсеновская модель** (`Rp_knudsen_areal`):
```
Rp(H) = Rp_skin + (R·T)/(M·D_eff)·H,   D_eff = (ε/τ)·(2/3)·r_pore·v̄
```
для cs ≈ 7 %, r_pore из режима заморозки.

**(в) Калибровка коэффициента теплопередачи** Kv = (×масштаб)·Kv_tubing — для
1-мл флакона на алюминиевом подносе (контакт + радиация плексигласовой камеры).

## 4. Откуда берутся расхождения

### 4.1 Дефолт vs калибровка (панель B)

| Версия | Kv | r_pore | Tc | t сушки | Tp_max | коллапс |
|---|---|---|---|---|---|---|
| Дефолт (как есть) | tubing ×1 | 8 мкм | −32 (Österberg) | 15.8 ч | −21 °C | ДА |
| **Калибровка** | **×2** (поднос) | **40 мкм** | **−20** (набл. Ronzi) | **8.9 ч** | −25 °C | нет |
| Ronzi (факт) | — | — | — | ~9 ч | — | нет |

Два физических вывода калибровки:

1. **Kv ≈ ×2 от Tang-Pikal (10cc tubing)** — это и закрывает расхождение по времени
   (15.8 → 8.9 ч). 1-мл флаконы на алюминиевом подносе имеют лучший тепловой
   контакт + получают радиацию от плексигласовой камеры.

2. **При Pc = 20 Па порог сублимации ≈ −31 °C** (там Pice = Pc): продукт физически
   не может быть холоднее ~−25 °C при рампе плиты до −5 °C. Раз Ronzi получили
   годный продукт при ~−25 °C, их **эффективная Tc ≈ −20 °C** (совпадает с их
   наблюдением «коллапс при −20»). Литературные −32 (Österberg, *другая* формуляция
   rFVIII) для концентрата Kedrion слишком консервативны.

### 4.2 Почему форма кривой здесь иная, чем в корневом `results.png`

Это **разные сценарии и разные величины**, а не ошибка:

| | корневой `results.png` (панель B) | здесь (панель A, первичная) |
|---|---|---|
| Ось Y | **Tp** (фронт сублимации) | **Tb** (дно флакона, ≈ термопара Ronzi) |
| Полка | **постоянная** → Tp выходит на **плато** (экспонента с насыщением) | **рампа** → продукт непрерывно растёт, с изломами на 4 и 8 ч |
| Pc / Kv / Rp | 0.1 Torr / ×1 / эмпир. | 0.15 Torr / ×2 / Кнудсен |

Tb всегда на 1–5 °C теплее Tp (между ними слой льда). А поднимающаяся плита тянет
продукт вверх, поэтому нет плато. Если построить тот же прогон величиной Tp при
постоянной полке — получится то же «экспоненциальное» семейство, что в `results.png`.

**Проверка этого — `verify_shapes.py` → `verify_shapes.png`:**
- панель A (полка рампится, режим Ronzi): Tp и Tb растут за плитой, плато нет;
- панель B (та же модель, полка постоянная): Tp выходит на плато — экспонента с
  насыщением, как в корневом `results.png`.

То есть это **одна и та же** `primary_drying()`; вся разница форм — от режима полки
(рампа vs const) и выбора величины (Tp фронта vs Tb дна), а не от разных моделей.
```bash
uv run python verify_shapes.py     # -> verify_shapes.png
```

---

## 5. Итог

- По **времени** (8.9 vs ~9 ч), **форме** (продукт ниже Tim, рост за плитой,
  скачок в конце) и **отсутствию коллапса** калиброванная модель ложится на Ronzi.
- Исходное расхождение объяснялось **некалиброванным Kv**, **оценкой Rp** и
  **неверно перенесённой Tc**; после калибровки модель согласуется с экспериментом.
- Заморозка и вторичная сушка на панели A — иллюстрация, не расчёт (модель их не
  покрывает).
